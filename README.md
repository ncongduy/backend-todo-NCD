# Backend Todo App

## Introduction

- Build api for user, todo
- Build authentication and authorization
- Use database PostgreSQL

## How to run

- Clone: `git clone git@github.com:ncongduy/backend-todo-NCD.git`
- Install depedencies: `npm install`
- Run command: `npm run dev`
